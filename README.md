# Next.js with Bun runtime

This is a [Next.js](https://nextjs.org/) project bootstrapped with Bun.

## Getting Started

### Cloning the repo

To run:

```bash
bun dev
```

On the browser:

[http://localhost:3000](http://localhost:3000)


## Deployed on Vercel

Currently deployed on [Vercel Platform](https://bun-nextjs-2.vercel.app/).
